# JS获取当前时间戳的3个方法

# 方法一

Date.now();

# 方法二

new Date().getTime();

# 方法三

+new Date();

关于方法三，需要了解一下一元正号+运算符的运行机制：

TIP

当+位于某操作数前面，目的是计算该操作数的数值，如果操作数不是一个数值，会尝试将其转换成一个数值，如果转换失败，将会得到NaN。所以背后的转换规则应该是使用Number(new Date())进行隐式转换，而Number()方法，当参数是对象时，则会遵循对象到原始值的转换规则转换为原始值，依次选择valueOf和toString进行转换。

看下MDN上的解释：

TIP

valueOf 方法返回以数值格式表示的一个 Date 对象的原始值，从1970年1月1日0时0分0秒（UTC，即协调世界时）到该日期对象所代表时间的毫秒数。 该方法的功能和 Date.prototype.getTime() 方法一样。
